## 🚀 خطوات النشر على GitHub Pages

1. **أنشئ مستودعاً جديداً** على [github.com](https://github.com/new)
2. **ارفع الملفات** (`index.html` + `README.md`) إلى المستودع
3. اذهب إلى **Settings → Pages**
4. تحت **Source** اختر: `Deploy from a branch`
5. اختر الفرع: `main` والمجلد: `/ (root)`
6. اضغط **Save**
7. انتظر دقيقة ثم افتح الرابط ✅
